# Omnistream

Watch multiple Twitch and YouTube streams in one place with a focus on your favorite POV.

<video src="assets/omnistream-showcase-2x.mp4" width="100%" controls autoplay loop muted></video>

### Use it here: [iredemption.github.io/omnistream/](https://iredemption.github.io/omnistream/)

### Why I made this

I worked on this primarily because most alternatives I've used didn't allow for resizing the bigger stream or didn't have an easy way to make one stream larger than the others. I took this as an opportunity to add features that would make it a nicer experience to watch multiple POV's of an event while being able to focus mostly on one person's perspective.

### Features

- Support for Twitch and YouTube streams.
- Ability to save and load groups of streams (e.g. FSLR, groups like Ludwig + Minecraft hatewatchers, etc).
- Select one stream to be larger than others and choose exactly how much bigger it should be.
- Option to show the chat of any added stream (Twitch and YouTube supported).
- Play and pause button for all streams simultaneously.

### Self-Hosting

You can also self-host this using Docker:

1. Clone the repo.
2. Run `docker-compose up -d`.
3. Open `http://localhost:5500` in your browser.
